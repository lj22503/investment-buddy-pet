# Investment Buddy Pet - 投资宠物技能

> 12 只宠物，1 个通用 Skill，通过配置文件区分宠物人格。

---

## 解决什么问题？

| 现状 | 用投资宠物 |
|------|----------|
| 客户亏钱了不知道怎么安抚 | 12 种宠物人格，不同风格的话术 |
| 不知道客户适合什么风格 | 20 题投资性格测试 → 推荐宠物 |
| 客户需要持续陪伴但顾不过来 | 宠物心跳引擎，自动触发陪伴任务 |
| 营销话术太生硬 | 宠物风格化表达，温暖/理性/果断/谨慎 |

## 4 个核心功能

| 功能 | 说明 |
|------|------|
| 投资性格测试 | 20 题五维测试（风险承受/知识/决策/情绪/时间偏好） |
| 宠物匹配 | 计算与 13 只宠物的兼容度，推荐 Top 3 |
| 宠物激活 | 选中宠物，配置心跳任务 |
| 心跳陪伴 | 情感陪伴 + 定投提醒 + 波动安抚 + 投资者教育 |

## 快速开始

```bash
# 安装
clawhub install investment-buddy-pet

# 运行性格测试
python3 scripts/personality_test.py

# 激活宠物
python3 scripts/heartbeat_engine.py start --user-id <your_user_id> --pet-type songguo
```

## FAQ

**Q：和投顾工作流有什么区别？**

A：投顾工作流专注"专业场景"（持仓诊断/资产配置），宠物专注"情感陪伴"，两者配合使用。

**Q：能推荐具体基金吗？**

A：不能。宠物只提供陪伴话术和情绪安抚，不给投资建议。

**Q：12 只宠物怎么选？**

A：先跑 personality_test.py，输出兼容度最高的 Top 3，选最喜欢的风格激活。
